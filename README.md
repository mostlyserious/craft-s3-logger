# mostlyserious/craft-s3-logger

## Installation

### composer.json
```json
"repositories": [
    {
        "type": "vcs",
        "url": "git@github.com:mostlyserious/craft-s3-logger.git"
    }
]
```

### Run
```shell
composer require mostlyserious/craft-s3-logger dev-master
```

```shell
php craft plugin/install s3-logger
```
